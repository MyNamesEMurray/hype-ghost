🎙 Voice awareness

Local mic transcription via the LocalVocal OBS plugin, so answering out loud is replying. It's third-party — keep it updated to match your OBS version (an outdated build can crash OBS, e.g. when switching Scene Collections). Optional: leave this Off to reply by typing.

🎧 Second channel — party / co-op audio

A separate audio device (Discord, TeamSpeak, a party call) transcribed by its own LocalVocal filter, so the cast can acknowledge the people you're playing with — kept distinct from your mic. In OBS, add a second LocalVocal Transcription filter to that device's source, output it to a different file or text source than your mic, and point this there.

Must be a different file/source than your mic's, or the two channels merge.

So the cast knows these voices aren't you — it can react to "your teammate" without mistaking them for the streamer.

👻 About Hype Ghost

Version … · GitHub · Release notes · Report a problem

A local-only cast of simulated AI viewers that watches your stream and chats with you — so you practice talking to chat even when nobody is watching. Updates install automatically from GitHub Releases (toggle on the App tab).

Every cast message is labeled 🤖 AI, and the overlay carries a permanent "simulated viewers, not real people" watermark — real viewers and VOD watchers are never misled. Nothing ever posts to your Twitch chat or affects stream metrics; all Twitch access is read-only.

Stream screenshots and (if enabled) your mic transcript go to the brain you chose — the Anthropic API, or your own local/OpenAI-compatible endpoint — to write messages, and nowhere else. Speech-to-text runs entirely locally inside OBS. With Twitch features on and no app credentials, your (public) channel name is sent to DecAPI. The app's server binds to 127.0.0.1 only. Your API key is stored unencrypted in the config file below — treat that folder like a password.

Files are copied to the new folder (the originals stay behind as a backup) and the app restarts. Handy for keeping logs and memory on another drive or in a synced folder.

Danger zone

Reset Hype Ghost to factory defaults: permanently deletes your settings (including the API key), session memory, chat history, and the cast's cross-stream profile, then restarts into the setup wizard.
